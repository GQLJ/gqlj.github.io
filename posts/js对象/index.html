《JS对象》

March 20, 2020

1.声明对象的两种语法

let obj = {'name':'xxx','age':'xxx'} //简单写法
let obj = new Object({'name':'xxx','age':'xxx'}) //正规写法
console.log =({'name':'xxx','age':'xxx'}) // 匿名写法

注:
1.{}本质上也是执行了new Object()
2.引号可以省略，但是省略后只能写标识符和数字，即使省略，键名也还是字符串
3.当你写对象的时候，'name' 这一部分永远是字符串
4.属性名（每个key都是对象的属性名）
  属性值（每个value都是对象的属性值）

2.如何删除对象的属性

  delete obj.xxx 
  delete obj['xxx']    //这两种写法可删除obj的xxx属性

  注意区分属性值为undefined和不含属性名

  obj.xxx = undefined
  obj.xxx = null

  上面只是把xxx的属性值变为undefined或null，xxx属性还保留着。

  使用'xxx' in obj 得到回复 false 或 true 判断是否删除成功

  代码示例：

  let obj = {
      'name' = 'jack'     
  }
  'name' in obj // true
  delete obj.name // 删除属性
  'name' in obj // false

3.查看对象的属性

  Object.keys(obj) // 查看自身所有属性
  Object.values(obj) //查看自身所有属性值
  Object.entries(obj) //同时查看自身所有属性和属性值
  console.dir(obj) //查看自身+共有的属性（控制台以目录形式打出来）

4.修改或增加对象的属性

已有则改，改只改自身属性。
没有则增，增只增自身属性。  ---不会影响到共有属性

代码示例：

let obj = {}
obj.name = 'jack' //添加name属性

let obj = { 'name':'jack'}
obj.name = 'rose' //修改name属性值

Object.assign()  //给对象批量添加属性

let obj = {}
Object.assign(obj,{
    'name':'jack',
    'age':'18'
})

5.‘name’ in obj 和 obj.hasOwnProperty(‘name’)的区别

in obj 不能区分自身属性和共有属性 使用该方法容易判断错误
hasOwnProperty可以判断是否有这个自身属性

代码示例：
let obj = {'name':'jack'}
'name' in obj // true
obj.hasOwnProperty('name') // true
'toString' in obj // true
obj.hasOwnProperty('toString') // false
