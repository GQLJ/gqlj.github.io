HTML 的发明

超文本标记语言（英语：HyperText Markup Language，简称：HTML) 由 Tim Berners-Lee 于 1990 年发明，是一种用于创建网页的标准标记语言。

HTML 起手式

  • 这三个标签是一个 HTML 文档的重要组成部分，缺一不可。

标签

1. 常用章节标签

  • <h1>-<h6> HTML 提供了6个标签，用来表示文章的标题。
  • <section> 表示一个含有主题的独立部分，通常用在文档里面表示一个章节。
  • <article> 表示页面里面一段完整的内容。
  • <main> 标签表示页面的主体内容，一个页面只能有一个。
  • <aside> 标签用来放置与网页或文章主要内容间接相关的部分。
  • <header>,<footer>,<p> 文档的头部、脚部、段落。

2. 内容标签

  1. <strong> 行内元素，浏览器会以粗体显示内容。
  2. <em> 行内标签，浏览器会以斜体显示它包含的内容。
  3. <code> 行内元素，浏览器默认会以等宽字体显示。
  4. <pre> 块级元素，浏览器会保留所有的换行和连续空格。
  5. <a> 链接通过<a>标签表示，浏览器会跳转到指定的网址

全局属性

所有元素都可以使用的属性

  • class 属性用来对网页元素进行分类。如果不同元素的 class 属性值相同，就表示它们是一类的。
  • contenteditable 布尔属性 允许用户修改浏览器内容。
  • hidden 布尔属性，浏览器不会出现这个元素。
  • id 属性在网页内的唯一标识符。比如，网页可能包含多个

    标签，id 属性可以指定每个

    标签的唯一标识符。

  • style 属性用来指定当前元素的 CSS 样式
  • tableindex 属性 定义了使用 Tab 键访问标签的顺序，其中 0 表示最后一个访问，正数从大到小依次访问，负数表示此标签不可访问。
  • title 属性 添加后鼠标悬停在该区域会有信息显示
