表达式和语句

什么是表达式和语句

  1. 表达式一般都有值，语句可能有可能没有。
    • 1+2 // 这是一个表达式
    • let a = 1+2 // 这是一个语句
    • let a = 1+2 ; let b = a //一个语句以分号结束，行末没有编译器会自动添加
  2. 语句一般会改变环境。（声明，赋值。）
  3. 表达式
    • 1+2 表达式的值为3。
    • add（1，2）表达式的值为函数的返回值。只有函数有返回值。
    • console.log也是一个表达式，他的值为函数本身。
    • console.log(3)表达式的值为：undefined。打印出的3是console打印出的一个记录。

  4. 以上两者并不绝对。

    标识符的规则

  5. 取名用到的就是标识符（区分大小写），也就是变量名。

  6. 常用的包括希腊字母 拉丁文 中文字符 “$“符号 下划线 “_” （这些都属于Unicode字母[即用双字节对字符进行编码；统一的字符编码标准]）

  7. 第一个字符后面的可以加数字，例如下面的：

    • a1
    • _abc
    • $123

    • 中文

      if else 语句

  8. 例1： if (表达式) {
    语句1
    } else {
    语句2
    }

  9. 例2： if(表达式){
    语句1
    }else if{
    语句2
    }else{
    语句3
    }

  10. 若分支只有一条，可用三元运算符表示。
    let a = b > 1 ? 1 : 0;
    if(a>1){ b=1 }else{ b=0 }
    这两个相同，花括号不能省略，不然只能作用到后面的第一个语句，空格无实际意义。

  11. 表达式为下列任意一种为假，其他情况为真。

  12. false

  13. undefined

  14. null

  15. 0

  16. “

  17. NaN

    while for 语句 （循环）

  18. while循环
    while（表达式）{
    循环体
    }

  19. for循环（while循环的方便写法）
    for（初始值；循环条件；递增表达式）{
    循环体 //满足条件后执行该条
    }

  20. 执行初始值，判断循环条件（条件不成立，退出for），执行循环体，再执行递增表达式。

    break和continue

  21. break退出整个循环。
    for (var i = 0; i <5; i++) {
    if (i === 3) {
    break;
    }
    console.log(i);
    } // 会打印 0, 1, 2

  22. continue退出当前一次循环。
    for (var i = 0; i < 5; i++) {
    if (i === 3) {
    continue;
    }
    console.log(i);
    } // 会打印 0, 1, 2, 4

    label

    “label”标签常配合break和continue使用，配合break使用可以跳出多层循环而不再局限于内层循环。
