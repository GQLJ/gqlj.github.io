CSS文档流

小科普: W3C规范中没有”文档流”（document flow）这个概念，只有normal-flow普通流
文档流叫法的由来是中文翻译问题。

  • 文档流的流动方向
    inline元素 从左往右 到达最后边才换行 一行容不下的时候 会折断跨两行
    block 元素 从上到下 每一个都独占一行
    inline-block元素 也是从左到右 但一行容不下的时候 不会跨行分成两块
    新的HTML5的标准里，元素是不分内联，块级的。

  • 关于宽度
    inline元素宽度由内部inline元素宽度的和决定的，不能设置宽度（width）。 block元素默认自动计算宽度（auto），可用 width 任意指定宽度。但永远不要写 width=“100%“，因为 auto 并不是 100% 。
    inline-block 元素结合了两者的特点 可用 width 指定。（即使比元素自身窄也可以）

  • 关于高度
    inline 元素的高度由line-height 间接决定，与height无关；
    block 元素的高度由内部文档流的元素决定，可以设置height；（如果内部什么都没有，高度为0）
    inline-block 元素与 block 元素类似，可以设置height。

  • 关于溢出
    当内容区大于容器设定的高度或宽度时，内容会溢出。
    可以用 overflow (溢出) 属性设置滚动条，拉动滚动条会显示溢出的内容。
    overflow: auto; （自动显示滚动条，内容区溢出时会出现，没有溢出时会消失。）
    overflow：scroll; （会一直显示滚动条，不论是否溢出。）
    overflow：hidden; （直接隐藏溢出部分，不再显示。）
    overflow: visible; （直接显示溢出部分。）

  • 脱离文档流
    block元素的高度由内部文档流的元素决定，可以设置height。这也表示有些元素可以不在文档流中，CSS设置了以下属性时，元素会脱离文档流。

    {
float：left/right/center;
}
{
position:absolute/fixed;
}

CSS盒模型

  • 盒模型是CSS中一个重要概念，它是元素大小的呈现方式。
    有句话说的好：“every element in web design is a rectangular box” 。
  • CSS有两种盒模型 border-box 和 content-box
    由四个区域组成，内容区（content）内边距（padding）边框（border）外边距（margin）
  • 两种盒模型之间的区别在于 border-box 的宽度包含 内容区 内边距 边框 外边距 而 content-box 的宽度只包含了 内容区
    这两种都可以通过 box-sizing 属性指定盒模型 但一般使用 border-box 较多。

CSS动画的两种做法

  • 每个静止的画面叫做”帧”，这些静止的画面连续播放起来便会形成动画。
    CSS动画是CSS3新增的功能，CSS通过 transform 属性控制HTML元素的变化，有两种方法可以设置动画，分别是transition 和 animation。

  • transform的四个常用功能 translate（位移），scale（缩放），rotate（旋转），skew（倾斜）。
    注： inline 元素不支持 transform ，要先变成block。

  • 动画是一个连贯的过程，transition 和 animation 可以为动画补充”中间帧”。
    transition语法：属性名 动画时长 过渡方式 延迟。
    可以用逗号分隔两个不同的属性，亦可用all表示所有属性。
    多次的过渡方式可以使用 animation 。
    animation语法：时长 过渡方式 延迟 次数 方向 填充模式 是否暂停 动画名 。

  • animation首先声明关键帧

    @keyframes xxx{             ---xxx是动画名<br>
0%{transform:none;}     ---这是一个关键帧：0%表示初始状态<br>
50%{transform:...}<br>
100{transform:...}      ---一共有三个关键帧。<br>
}

  • 添加动画

    #demo.start{
animation:xxx 1s;            ---给demo的start属性添加动画。
}
button.onclick =()=>{
demo.classList.add('start')  ---点击按钮，则给demo添加一个  start属性。
}

    一般会使用功能更加强大的 animation。
